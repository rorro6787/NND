Algorithm Benchmark Toolkit
===========================

This is the documentation for the `neuro_disease_detector` project, which
aims to provide tools for benchmarking algorithms in machine learning. 

Installation
============

To install the project, you can use the following command:

```bash
pip install neuro_disease_detector
```
